## Setup local project

```bash
# cloning this project
git clone https://github.com/kochan4php/kotree.git

# go to the project folder
cd kotree

# install pnpm if you don't have it
npm install -g pnpm

# install dependencies
pnpm install

# run server app
pnpm dev
```
